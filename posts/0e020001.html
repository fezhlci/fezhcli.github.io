高中化学常用化学方程式

发表于2022-03-19|更新于2023-03-21
|字数总计:585|阅读时长:2分钟|阅读量:

无机化学

注意：请严格按照化学方程式规则写法书写。因技术原因，本博客目前只能使用连续等号代替长等号。

1. 钠在空气中燃烧（或加热）

2Na+O2===燃烧Na2O22Na + O_2 \stackrel{\text{燃烧}}{===} Na_2O_2 2Na+O2​===燃烧Na2​O2​

2Na+O2===ΔNa2O22Na + O_2 \stackrel{\Delta}{===} Na_2O_2 2Na+O2​===ΔNa2​O2​

2. 钠与水反应

2Na+2H2O==2NaOH+H2↑2Na + 2H_2O == 2NaOH + H_2↑ 2Na+2H2​O==2NaOH+H2​↑

2Na+2H2O==2Na++OH−+H2↑2Na + 2H_2O == 2Na^+ + OH^- + H_2↑ 2Na+2H2​O==2Na++OH−+H2​↑

3. 过氧化钠与水反应

2Na2O2+2H2O==4NaOH+H2↑2Na_2O_2 + 2H_2O == 4NaOH + H_2↑ 2Na2​O2​+2H2​O==4NaOH+H2​↑

2Na2O2+2H2O==4Na++OH−+H2↑2Na_2O_2 + 2H_2O == 4Na^+ + OH^- + H_2↑ 2Na2​O2​+2H2​O==4Na++OH−+H2​↑

4. 过氧化钠与二氧化碳反应（过氧化钠做供氧剂的原理）

2Na2O2+2CO2==2Na2CO3+O22Na_2O_2 + 2CO_2 == 2Na_2CO_3 + O_2 2Na2​O2​+2CO2​==2Na2​CO3​+O2​

5. 向盐酸中滴入碳酸钠溶液

Na2CO3+2HCl===2NaCl+HO+CO2↑Na_2CO_3 + 2HCl === 2NaCl + H_O + CO_2↑ Na2​CO3​+2HCl===2NaCl+HO​+CO2​↑

6. 向碳酸钠溶液中滴入盐酸

Na2CO3+HCl===NaCl+NaHCO3Na_2CO_3 + HCl === NaCl + NaHCO_3 Na2​CO3​+HCl===NaCl+NaHCO3​

7. 碳酸氢钠（小苏打）受热分解

2NaHCO3===ΔNa2CO3+H2O+CO2↑2NaHCO_3 \stackrel{\Delta}{===} Na_2CO_3 + H_2O + CO_2↑ 2NaHCO3​===ΔNa2​CO3​+H2​O+CO2​↑

8. 固体氢氧化钠与碳酸氢钠共同加热

NaHCO3+NaOH===ΔNa2CO3+H2ONaHCO_3 + NaOH \stackrel{\Delta}{===} Na_2CO_3 + H_2O NaHCO3​+NaOH===ΔNa2​CO3​+H2​O

9. 硫酸氢钠的分解

NaHSO4=Na++H++SO42−NaHSO_4 = Na^+ + H^+ + SO_4^{2-} NaHSO4​=Na++H++SO42−​

NaHSO4===ΔNa++HSO4−NaHSO_4 \stackrel{\Delta}{===} Na^+ + HSO_4^- NaHSO4​===ΔNa++HSO4−​

10. 钠单质的制备

2NaCl（熔融）===电解2Na+Cl22NaCl{\text{（熔融）}}\stackrel{\text{电解}}{===} 2Na + Cl_2 2NaCl（熔融）===电解2Na+Cl2​

11. 氯碱工业（电解饱和食盐水）

2NaCl+2H2O===电解2NaOH+H2↑+Cl2↑2NaCl + 2H_2O \stackrel{\text{电解}}{===} 2NaOH + H_2↑ + Cl_2↑ 2NaCl+2H2​O===电解2NaOH+H2​↑+Cl2​↑

12. 侯氏制碱法

总反应：

NaCl+NH3⋅H2O+O2===NH4Cl+NaHCO3(s)NaCl + NH_3·H_2O + O_2 === NH_4Cl + NaHCO_{3(s)} NaCl+NH3​⋅H2​O+O2​===NH4​Cl+NaHCO3(s)​

分步反应：

NH3+H2O⇌NH3⋅H2ONH_3 + H_2O {\rightleftharpoons} NH_3·H_2O NH3​+H2​O⇌NH3​⋅H2​O

CaCO3===高温CO2+CaOCaCO_3 \stackrel{\text{高温}}{===} CO_2 + CaO CaCO3​===高温CO2​+CaO

NH3⋅H2O+CaO===NH4HCO3NH_3·H_2O + CaO === NH_4HCO_3 NH3​⋅H2​O+CaO===NH4​HCO3​

NaCl+NH4HCO3===NaHCO3↓+NH4ClNaCl + NH_4HCO_3 === NaHCO_3↓ + NH_4Cl NaCl+NH4​HCO3​===NaHCO3​↓+NH4​Cl

13. 锂单质在空气中燃烧

4Li+O2===点燃2Li2O4Li + O_2 \stackrel{\text{点燃}}{===} 2Li_2O 4Li+O2​===点燃2Li2​O

14. 镁条在空气中燃烧

2Mg+O2===点燃2MgO2Mg + O_2 \stackrel{\text{点燃}}{===} 2MgO 2Mg+O2​===点燃2MgO

3Mg+N2===点燃Mg3N23Mg + N_2 \stackrel{\text{点燃}}{===} Mg_3N_2 3Mg+N2​===点燃Mg3​N2​

2Mg+CO2===点燃2MgO+C2Mg + CO_2 \stackrel{\text{点燃}}{===} 2MgO + C 2Mg+CO2​===点燃2MgO+C
